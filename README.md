# Интерактивная схема CPO

Самодостаточная HTML/CSS/JS-схема по образцу изображения: центральные платформы Telecom, CX и VAS, бизнес-связи, принципы управления, интеграционный слой, бизнес-юниты и RACI.

## Как открыть

Откройте `index.html` в браузере или запустите любой статический сервер в корне репозитория.

## Возможности

- кликабельные блоки с панелью деталей;
- подсветка связанных элементов;
- поиск по названиям, владельцам и описаниям;
- адаптивная верстка для широких и узких экранов.
