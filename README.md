# Интерактивная схема CPO на Canvas

Самодостаточная HTML/CSS/JS-схема по образцу изображения. Основная схема отрисована на HTML5 Canvas: центральные платформы Telecom, CX и VAS, бизнес-связи, принципы управления, интеграционный слой, бизнес-юниты и RACI.

## Как открыть

Откройте `index.html` в браузере или запустите любой статический сервер в корне репозитория.

## Возможности

- отрисовка схемы на `<canvas>`;
- кликабельные Canvas-блоки с панелью деталей;
- подсветка связанных элементов прямо на холсте;
- поиск по названиям, владельцам и описаниям;
- адаптивная верстка для широких и узких экранов.
