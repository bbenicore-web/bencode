# Импорт схемы CPO в Canva

## Быстрый способ

1. Открой Canva и создай дизайн с размером **1920 × 1400 px** (или **Presentation 16:9**).
2. Загрузи файл `cpo-scheme-canvas.png` из этой папки.
3. Растяни изображение на весь слайд.
4. При необходимости добавь поверх редактируемые текстовые блоки по файлу `cpo-content.json`.

## Файлы

- `cpo-scheme-canvas.png` — чистый экспорт схемы без поиска и панелей
- `cpo-scheme-page.png` — скрин всей страницы с интерфейсом
- `cpo-content.json` — структура блоков, владельцев и RACI для ручной доработки в Canva

## Если нужен автоматический перенос через Cursor

Canva MCP в этом окружении сейчас не авторизован. Чтобы я создал дизайн прямо в Canva:

1. Открой Cursor Desktop.
2. Перейди в **Settings → MCP → Canva**.
3. Авторизуй Canva.
4. Напиши: «Создай Canva-дизайн из `/workspace/canva/cpo-scheme-canvas.png`».

После авторизации я смогу загрузить PNG в Canva и оформить слайд автоматически.
